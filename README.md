# swe-testing-assignment
# Quick-Calc Calculator

This project implements a simple calculator application written in Python.

The calculator supports the following operations:

- Addition
- Subtraction
- Multiplication
- Division

The project also includes automated tests using **pytest** to verify correctness.

---

# Setup Instructions

1. Clone the repository

git clone https://github.com/maha-862006/swe-testing-assignment.git

2. Navigate to the folder

cd swe-testing-assignment

3. Install pytest

pip install pytest

---

# Running the Application

Run the calculator application using:

python app.py

---

# Running Tests

Execute the test suite using:

pytest

All tests should pass successfully.

---

# Testing Framework Comparison

This project uses **pytest** because it is simple and requires less boilerplate code compared to **unittest**.

Advantages of pytest:

- simpler syntax
- automatic test discovery
- powerful fixtures
- better readability

Unittest is built into Python but requires more structure and setup.
